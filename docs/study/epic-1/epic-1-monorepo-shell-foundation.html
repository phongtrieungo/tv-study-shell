TV Study Shell · Epic 1 synthesis

Monorepo Shell Foundation

One start→end picture of Epic 1: how a Vite/pnpm monorepo becomes a TV-shaped Shell that can mount and unmount Surfaces with cleanup — before Canvas, WebGL, Blits, or Solid Labs arrive.

The problem this epic solves

Interview stacks (Lightning/Blits, Solid, Canvas, WebGL, Smart TV constraints) do not stick as slideware. You need a single runnable app where navigation chrome, shared data/input, and Surface lifecycle are real — so later Labs plug into a contract instead of inventing their own shell each time.

  • FR-1 / FR-3: Launch a Safe Zone Shell and choose a Surface.
  • FR-2 / AD-2 / AD-6: Switch Surfaces via documented mount/unmount and dispose side effects.
  • AD-3 / AD-4: One Mock Data plane and one D-pad map for every Surface.

Learning arc (1.1 → 1.4)

≥50 channels, ≥24h programs, Home rail ≥12 items; KEY / getDpadAction. Vocabulary every Lab will share. Study HTML DoD begins here.

Deep dive: 1-2 study guide

Visible ~5% Safe Zone; focusable Home / Live / EPG / WebGL Lab. Select acknowledged only — Select ≠ mount yet. Thin DOM Shell owns chrome.

Deep dive: 1-3 study guide

1.4 — Host mount/unmount contract

Stub Surface package; Shell host registry; Enter mounts, Back unmounts; listener/timer cleared (hasActiveSideEffects()). Error banner on mount failure.

Deep dive: 1-4 study guide

How the pieces fit

apps/shell          thin DOM chrome + menu + Surface host
packages/shared     fixtures + D-pad map + types
packages/surface-stub   proves mount/unmount cleanup (temporary)

menu → select → host.enter(id) → module.mount(hostEl)
Back → host.leave() → module.unmount() → menu

All four menu ids currently resolve to the stub via a registry. Epic 2 swaps epg to @tvshell/epg-canvas without rewriting Shell chrome.

Demo / interview script (≈ 3 minutes)

  1. pnpm install && pnpm dev → open http://localhost:5180.
  2. Point at the dashed Safe Zone; explain overscan vs portfolio 5% convention.
  3. Arrow through destinations; name the shared D-pad map (not ad-hoc key codes).
  4. Enter → stub mounts; note the cleanup probe ticking.
  5. Escape/Back → unmount; status says side effects cleared; probe is false.
  6. Whiteboard: Shell owns enter/leave; Surface disposes what it created (AD-6).
  7. Honesty: desktop Chromium + keyboard proxy; stub ≠ Memory Soak; real Labs arrive next.

Honesty bounds

  • No real EPG / WebGL / Home / Live packages yet — only the stub.
  • Vitest / Playwright deferred to Epic 7; quality gate here was typecheck + build + manual smoke.
  • Blits iframe vs embed mount strategy still deferred to Story 4.1.

What Epic 2 builds on

Canvas EPG needs: shared fixtures for the grid, shared D-pad for focus, and the host contract so leave disposes RAF/canvas resources the same way the stub cleared its timer/listener.
