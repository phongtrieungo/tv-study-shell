One start→end picture of Lab W: the same Visible Window idea as Canvas EPG, rendered with raw WebGL — then measured against Canvas and explained with honest pipeline vocabulary so you can answer “WebGL or only frameworks?” without overclaiming.

The problem this epic solves

Canvas EPG (Epic 2) proved virtualization. Interviewers still ask about buffers, textures, shaders, draw calls and GPU vs CPU trade-offs. Lightning/Blits can hide the metal. Epic 3 closes the gap with a first-class raw WebGL Surface that reuses shared Visible Window math, then documents measured compare + vocabulary at learning-lab depth.

Demo / interview script (~5 minutes)

  1. Shell → EPG → HUD drawn ≪ logical 600 (Canvas virtualization).
  2. Back → WebGL Lab → HUD + context webgl2; arrows move the same window idea on the GPU.
  3. Open README → WebGL vocabulary: name buffer / texture / shader / draw call → point at lab symbols; say one batched drawArrays.
  4. Open docs/perf-notes/canvas-vs-webgl.md — same machine, labeled env; refuse “WebGL always wins” / unlabeled 60 FPS.
  5. Honesty close: learning lab depth; Blits/Lightning Home (Epic 4) is applied WebGL next — not a redo of Lab W.

Whiteboard talking points

  • Same math, different renderer — windowing is data; Canvas vs WebGL is a renderer choice.
  • Four terms — buffer (createVbo / bufferData), texture (texImage2D atlas), shader (VS/FS program), draw call (one drawArrays).
  • Dispose — disposeGpu; VRAM is not collected by JS GC.
  • Measure → claim — AD-8 side-by-side note; desktop proxy ≠ OEM TV.
  • Not Blits yet — Lab W teaches metal; Home teaches scene graph + texture lifecycle.

Honesty bounds

  • Desktop Chromium + keyboard-as-D-pad — not Tizen/webOS certification.
  • Learning lab / textured 2D UI tiles — not graphics-engine expertise.
  • Stage size changes drawn counts (same virtualization, different viewport).
  • No drawElements in this lab — do not invent indexed draws on the whiteboard.
  • Vitest / Playwright still Epic 7 — confidence today is typecheck + build + smoke + docs.
  • HiDPI / Date.parse / overlapping nav — still deferred; do not expand into Epic 4 unless blocked.
